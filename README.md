# Simple Interest Calculator

## Description
This project calculates simple interest using the formula:

Simple Interest = (P × R × T) / 100

Where:
P = Principal amount
R = Rate of interest
T = Time

## Example
If P = 1000, R = 5%, T = 2 years:
SI = (1000 × 5 × 2) / 100 = 100
